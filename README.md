# Zero to Master - React Course Projects

Colección de proyectos del curso de React "Zero to Master".

## Proyectos incluidos

- **counter-app** - App de contador con tests
- **gifexpertapp** - Buscador de GIFs
- **heros-app** - App de héroes con autenticación
- **hook-app** - Ejemplos de React Hooks
- **journal-chill-app** - App de diario
- **redux-journal** - Diario con Redux
- **my-app** - Ejercicios básicos

## Instalación

Cada proyecto tiene su propio `package.json`:

```bash
cd [nombre-proyecto]
npm install
npm start
```

## Temas cubiertos

- useState, useEffect, useContext
- useReducer, useMemo, useCallback
- React Router
- Redux
- Testing con Jest
- Autenticación
- Firebase

## Propósito

Proyectos de práctica del curso de React de Fernando Herrera.
